
# Airline Passenger Satisfaction Classification

## Overview

This project focuses on analyzing and predicting airline passenger satisfaction using machine learning. The goal is to classify whether a passenger is satisfied or not based on many features related to flight experience and service quality.

---

## Problem

Airline companies wants to improve customer experience by understanding satisfaction levels. This project aims to predict passenger satisfaction using data-driven approaches.

---

## Approach

### Data Preprocessing

* Handled missing values
* Selected relevant features

### Exploratory Data Analysis

* Analyzed feature distributions
* Visualized relationships between variables
* Identified patterns affecting satisfaction

### Machine Learning

* Trained classification models
* Evaluated model performance using accuracy and other metrics

---

## Results

The model was able to classify passenger satisfaction with good accuracy, showing that features such as service quality and travel class significantly impact satisfaction.
